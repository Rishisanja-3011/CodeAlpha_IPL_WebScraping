# IPL Player Statistics Analysis

## Overview

This project performs exploratory data analysis on IPL cricket datasets using Python.

The analysis focuses on:
- Player batting performance
- Strike rate comparisons
- Six-hitting ability
- Toss decision trends
- Team winning statistics

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
